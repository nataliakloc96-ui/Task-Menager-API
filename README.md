# Task Manager API

Backend API napisane w FastAPI.

## Technologie

* FastAPI
* SQLAlchemy
* PostgreSQL
* Redis
* JWT Authentication
* Docker
* Pytest

## Funkcjonalności

* Rejestracja użytkownika
* Logowanie
* Access Token
* Refresh Token
* CRUD zadań
* Filtrowanie zadań
* Rate Limiting

## Uruchomienie

```bash
docker compose up --build
```

## Testy

```bash
pytest
```

## Dokumentacja API

Po uruchomieniu:
