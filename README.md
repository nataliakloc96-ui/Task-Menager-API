# Task Manager API

![Tests](https://github.com/nataliakloc96-ui/Task-Menager-API/actions/workflows/tests.yml/badge.svg)

## Technologie

* FastAPI
* SQLAlchemy
* PostgreSQL
* Redis
* JWT Authentication
* Docker
* Pytest

## Funkcjonalności

* Rejestracja użytkownika
* Logowanie
* Access Token
* Refresh Token
* CRUD zadań
* Filtrowanie zadań
* Rate Limiting

## Uruchomienie

```bash
docker compose up --build
```

## Testy

```bash
pytest
```
